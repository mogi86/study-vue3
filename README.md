# study-vue3

## Requirements

- Node 14
- GO 1.17

## Run API Server

```
cd go-api-server
go run main.go
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
